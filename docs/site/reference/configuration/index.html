Configuration¶

This page summarizes the important project configuration files.

Dart And Flutter Package¶

pubspec.yaml defines the package:

name: acta
description: "A new Flutter project."
publish_to: 'none'
version: 0.1.0+1

The SDK constraint is:

environment:
  sdk: ^3.12.2

Runtime dependencies:

dependencies:
  flutter:
    sdk: flutter

Development dependencies:

dev_dependencies:
  flutter_test:
    sdk: flutter
  flutter_lints: ^6.0.0

The app currently enables Material icons and default Material asset support:

flutter:
  uses-material-design: true

Analyzer¶

analysis_options.yaml controls static analysis. The project currently uses the standard Flutter lint baseline generated by Flutter.

Documentation Site¶

docs/mkdocs.yml configures MkDocs with:

  • Site name: Acta Documentation.
  • Theme: readthedocs.
  • Navigation for overview, setup, development, and reference pages.
  • Markdown extensions for admonitions and permalinked headings.

Run the docs locally with:

mkdocs serve -f docs/mkdocs.yml

Build the docs with:

mkdocs build -f docs/mkdocs.yml
